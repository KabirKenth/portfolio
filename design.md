# Personal Website Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a1f36) - Professional, trustworthy foundation
- **Secondary**: Warm Copper (#c17817) - Accent color for highlights and CTAs  
- **Tertiary**: Soft Sage (#8fbc8f) - Subtle accent for secondary elements
- **Neutral**: Warm White (#faf9f6) - Clean background and text contrast

### Typography
- **Display Font**: "Tiempos Headline" - Bold serif for hero headings and major titles
- **Body Font**: "Suisse Int'l" - Clean sans-serif for readability and modern feel
- **Code Font**: "JetBrains Mono" - Technical projects and code snippets

### Visual Language
- **Minimalist Sophistication**: Clean layouts with purposeful white space
- **Technical Precision**: Sharp edges, geometric patterns reflecting coding background
- **Warm Professionalism**: Balanced color palette that feels approachable yet expert
- **Data-Driven Aesthetic**: Subtle grid patterns and chart-like visual elements

## Visual Effects & Animation

### Hero Background Effects
- **Animated Shader**: Liquid-metal displacement effect using shader-park
- **Particle System**: Subtle floating particles representing data points
- **Gradient Flow**: Slow-moving aurora-like gradients in brand colors

### Text Effects
- **Typewriter Animation**: Name appears with typing effect using Typed.js
- **Color Cycling**: Key skills highlight with rotating accent colors
- **Split-letter Stagger**: Section headings animate in letter by letter

### Interactive Elements
- **3D Tilt**: Project cards lift and tilt on hover
- **Glow Edges**: Buttons and links get subtle glow effects on interaction
- **Morphing Shapes**: Background geometric shapes transform smoothly
- **Depth Shadows**: Dynamic shadows that respond to mouse movement

### Data Visualization Style
- **Skills Radar**: Animated radar chart with smooth transitions
- **Project Metrics**: Clean bar charts with brand color scheme
- **Timeline Navigation**: Horizontal timeline with interactive nodes

### Scroll Motion
- **Parallax Layers**: Subtle depth effect with background elements
- **Reveal Animations**: Content fades in as user scrolls (16-24px translation)
- **Stagger Delays**: Sequential animation of cards and list items

### Hover Effects
- **Card Elevation**: Project cards lift with expanding shadows
- **Image Zoom**: Portfolio images scale slightly on hover
- **Button Morphing**: CTA buttons change color and add glow
- **Icon Animation**: Social icons rotate or scale on hover

## Layout Principles
- **Grid System**: 12-column responsive grid with consistent spacing
- **Asymmetrical Balance**: Dynamic layouts that feel natural yet structured
- **Content Hierarchy**: Clear visual hierarchy using typography and spacing
- **Mobile-First**: Touch-friendly interactions and responsive design
