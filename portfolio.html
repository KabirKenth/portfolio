FreightSwipe

Project Overview

FreightSwipe is a comprehensive logistics platform that modernizes freight matching through intuitive swipe-based interactions. The application connects shippers with available carriers in real-time, featuring advanced matching algorithms and comprehensive review systems.

Key Features

  • • Swipe-based load matching interface for intuitive user experience
  • • Real-time dashboard with live load updates and carrier availability
  • • JWT-based authentication ensuring secure user sessions
  • • Two-way review system building trust between shippers and carriers
  • • RESTful API architecture for scalable service integration
  • • Docker containerization for consistent deployment across environments

Technical Implementation

Built with React for the frontend user interface, Node.js and Express for the backend API, and PostgreSQL with Prisma ORM for efficient data management. The application implements secure JWT authentication and is fully containerized using Docker and Docker Compose for seamless deployment.

Impact & Results

This project demonstrates full-stack development expertise, combining modern web technologies with practical logistics solutions. The swipe-based interface improves user engagement while the real-time matching system enhances operational efficiency for freight management.

Walmart Data Scraper

Project Overview

An advanced web scraping system designed for automated collection of product data from Walmart's e-commerce platform. The system handles large-scale data extraction with intelligent error handling and scalable database architecture.

Technical Features

  • • BeautifulSoup and Requests for efficient web scraping
  • • Intelligent retry logic with exponential backoff
  • • Rate limiting to prevent IP blocking and ensure respectful scraping
  • • SQLite database with optimized indexing for fast queries
  • • PostgreSQL migration strategy for production scaling
  • • Docker Compose configuration for modular deployment

Data Collection

The scraper collects comprehensive product information including product names, pricing data, customer ratings, availability status, and product descriptions. All data is structured and stored with proper normalization for analytical purposes.

Architecture & Reliability

Built with reliability and scalability in mind, the system implements comprehensive error handling, automatic retry mechanisms, and database optimization strategies. The containerized deployment ensures consistent performance across different environments.
